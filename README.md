# marketplace
CEL.ro Marketplace API wrapper in PHP

___
## Install 
>composer.json
```
{  
    "require": {  
        "celdotro/marketplace": "*"
    },
    "minimum-stability": "dev"
}
```

>Command
```
composer install
```
___
## Confiugre
**/vendor/celdotro/marketplace/src/Dispatcher.php** => change userName and password constants
